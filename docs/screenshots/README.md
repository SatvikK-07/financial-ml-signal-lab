# Dashboard Screenshot Placeholders

Capture these screenshots after deploying or refreshing market data so the
visible dates and freshness labels are accurate:

1. `overview-trust.png` — Overview signal card and “Can I trust this signal?”
   breakdown.
2. `live-market.png` — Live Market candlestick chart with freshness and zones.
3. `paper-trading.png` — Paper Trading setup, journal, and summary metrics.
4. `research-backtest.png` — Research Mode backtest and execution-policy view.

Screenshots should not imply live broker execution or guaranteed profitability.
